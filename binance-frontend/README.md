# Semantic Versioning

2.4.3 -> 2.4.4 -> 2.4.5

2.4.1 -> 2.5.3

2.4.1 -> 3.1.2

2: Major Release -> Breaking Compatibility 

4: Minor Release -> Backward Compatible, New Features

3: Patch Release -> Backward Compatible, No new Feature

~ -> Accept Patch Release

^ -> Accept Minor Release

~2.4.3

^2.4.3 -> Accepts 2.5.1
